# User Roles and Features

## Overview

The Internship Management Platform is designed to connect students seeking internship opportunities with companies looking to hire interns. The platform streamlines the internship application process by allowing students to create profiles, apply for internships, and track their applications, while enabling companies to post internship opportunities and manage applicants.

---

# User Roles

## 1. Student

### Description

Students are the primary users of the platform who search for internship opportunities and apply to positions that match their skills and interests.

### Responsibilities

* Create and maintain a professional profile
* Upload and update resumes
* Browse available internships
* Submit internship applications
* Track application status

### Goals

* Discover relevant internship opportunities
* Showcase skills and qualifications
* Secure internship positions

---

## 2. Company

### Description

Companies use the platform to advertise internship opportunities and recruit qualified students.

### Responsibilities

* Create and maintain company profiles
* Post internship opportunities
* Review student applications
* Evaluate candidate qualifications
* Accept or reject applicants

### Goals

* Reach qualified student candidates
* Simplify the recruitment process
* Fill internship positions efficiently

---

# User Features

## Student Features

### 1. Authentication

Students can register and securely access the platform.

#### Functionalities

* Sign Up
* Login
* Logout
* Password Recovery

---

### 2. Profile Management

Students can create and manage their professional profiles.

#### Profile Information

* Full Name
* Email Address
* Phone Number
* College/University
* Degree Program
* Graduation Year
* Skills

#### Resume Management

* Upload Resume (PDF format)
* Update Resume
* View Uploaded Resume

---

### 3. Browse Internships

Students can explore available internship opportunities posted by companies.

#### Internship Information Displayed

* Internship Title
* Company Name
* Location
* Duration
* Stipend
* Required Skills
* Application Deadline
* Internship Description

---

### 4. Apply for Internships

Students can submit applications for internships directly through the platform.

#### Application Process

1. Browse internship listings
2. View internship details
3. Click the Apply button
4. Submit application with attached resume

---

### 5. Track Application Status

Students can monitor the progress of their applications.

#### Application Statuses

* Applied
* Under Review
* Accepted
* Rejected

#### Information Available

* Internship Applied For
* Company Name
* Application Date
* Current Status

---

# Company Features

## 1. Authentication

Companies can create accounts and securely access the platform.

#### Functionalities

* Sign Up
* Login
* Logout

---

## 2. Company Profile Management

Companies can maintain organizational information.

#### Profile Information

* Company Name
* Industry
* Website
* Location
* Company Description

---

## 3. Post Internship Opportunities

Companies can create internship listings to attract candidates.

#### Internship Details

* Internship Title
* Description
* Required Skills
* Duration
* Stipend
* Location
* Application Deadline

---

## 4. Manage Internship Listings

Companies can manage all internship opportunities they have posted.

#### Functionalities

* Create Internship Posting
* Edit Internship Posting
* Delete Internship Posting
* View Active Listings

---

## 5. View Applications

Companies can review applications submitted by students.

#### Applicant Information Available

* Student Name
* Educational Background
* Skills
* Resume

---

## 6. Candidate Selection

Companies can evaluate candidates and make hiring decisions.

#### Available Actions

* Accept Candidate
* Reject Candidate

Once a decision is made, the application status is automatically updated and becomes visible to the student.
